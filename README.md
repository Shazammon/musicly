# ScaleSkills

ScaleSkills is a fullstack app where music students looking for remote education can find highly rated instructors for their instrument and schedule lessons that fit with their schedule. 

# Tech Stack

* PostgresQL
* Django
* Next.js
* Tailwind CSS
